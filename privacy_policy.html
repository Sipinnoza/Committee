Agentra（群策）隐私政策

生效日期：2026年4月21日

1. 概述

Agentra（群策）是一款多 Agent 自主讨论与决策应用。我们非常重视您的隐私。本隐私政策说明了应用如何处理您的数据。

2. 数据收集与使用

我们不收集任何个人数据。 具体而言：

  • 应用不要求注册或登录
  • 应用不收集姓名、邮箱、电话号码等个人身份信息
  • 应用不收集设备标识符或位置信息
  • 应用不包含任何分析或追踪 SDK

3. API 密钥存储

应用需要您提供第三方 LLM 服务的 API 密钥才能正常运行。关于 API 密钥：

  • API 密钥仅存储在您设备本地（使用 Android DataStore）
  • API 密钥不会上传到我们的服务器或任何第三方（除对应的 LLM 服务提供商外）
  • 您可以随时在应用设置中删除已保存的 API 密钥

4. 第三方服务

应用会将您在会议中的讨论内容发送至以下第三方 LLM API 以获取 AI 响应：

  • Anthropic Claude API
  • DeepSeek API
  • Moonshot (Kimi) API
  • GLM (智谱) API

发送的数据仅包含会议讨论上下文文本，不包含设备信息或个人身份数据。各服务的数据处理方式请参阅其各自的隐私政策。

5. 本地数据存储

以下数据仅存储在您的设备本地：

  • 会议记录和讨论历史（Room 数据库）
  • 应用设置和偏好（DataStore）
  • API 密钥配置（DataStore）

卸载应用将永久删除所有本地数据。

6. 网络搜索功能

应用包含可选的网络搜索功能，用于在会议讨论中提供参考信息。搜索查询会发送至您配置的搜索服务，不会附带个人身份信息。

7. 儿童隐私

本应用不面向 13 岁以下儿童，也不会有意收集儿童的任何信息。

8. 隐私政策变更

我们可能会不时更新本隐私政策。任何变更将在本页面发布，并更新生效日期。

9. 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：

Agentra Privacy Policy

Effective Date: April 21, 2026

1. Overview

Agentra is a multi-Agent autonomous discussion and decision-making application. We take your privacy seriously. This privacy policy explains how the app handles your data.

2. Data Collection and Use

We do not collect any personal data. Specifically:

  • The app does not require registration or login
  • The app does not collect names, emails, phone numbers, or other personally identifiable information
  • The app does not collect device identifiers or location data
  • The app does not include any analytics or tracking SDKs

3. API Key Storage

The app requires you to provide API keys for third-party LLM services to function. Regarding API keys:

  • API keys are stored only locally on your device (using Android DataStore)
  • API keys are never uploaded to our servers or any third party (except the corresponding LLM service provider)
  • You can delete saved API keys at any time from the app settings

4. Third-Party Services

The app sends meeting discussion content to the following third-party LLM APIs to obtain AI responses:

  • Anthropic Claude API
  • DeepSeek API
  • Moonshot (Kimi) API
  • GLM (Zhipu) API

Only meeting discussion context text is sent. No device information or personal identity data is included. Please refer to each service's privacy policy for their data handling practices.

5. Local Data Storage

The following data is stored only on your device:

  • Meeting records and discussion history (Room database)
  • App settings and preferences (DataStore)
  • API key configurations (DataStore)

Uninstalling the app will permanently delete all local data.

6. Web Search Feature

The app includes an optional web search feature to provide reference information during meeting discussions. Search queries are sent to your configured search service without any personally identifiable information.

7. Children's Privacy

This app is not directed at children under 13 and does not knowingly collect any information from children.

8. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated effective date.
