群策 Agentra 是一款多 Agent 自主讨论与决策应用。六位专业 AI 智能体——分析师、风控官、策略师、情报员、执行者、督导——围绕您关注的投资标的，展开结构化的投资委员会会议。

核心特性：
- 六位 Agent 自主讨论：每位 Agent 拥有独立的专业视角和决策逻辑
- 多轮辩论与投票：Agent 之间会质疑、补充、反驳，最终形成加权投票评级
- 支持多种 LLM：Anthropic Claude、DeepSeek、Moonshot (Kimi)、GLM (智谱)
- 会议全程记录：完整保存讨论过程、投票结果和最终评级
- 智能搜索增强：可选联网搜索，让 Agent 获取最新市场信息
- 自进化机制：Agent 在会后进行自我反思，持续优化决策能力
- 黑板协作系统：Agent 通过共享黑板进行信息交换和协作
- 稀疏激活：智能选择最相关的 Agent 参与讨论，提升效率

完全离线运行，数据存储在本地。您只需配置自己的 LLM API 密钥即可使用。

适合投资研究、决策分析、以及对 Multi-Agent 系统感兴趣的开发者和研究者。
